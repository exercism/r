# Design

## Goal

The goal of this exercise is to introduce errors and (very basic) logging in R.

## Learning objectives

- Understand how to use `stop()` and `stopifnot()` for serious errors.
- Know about the logging functions `warning()` and `message()`.

## Out of scope

- Error handling with `try()` and `tryCatch()`.
- These more advanced methods were considered for inclusion, but they are rarely if ever used in Exercism solutions.
  They are mentioned in the concept `about.md`, but not this exercise.

## Concepts

This exercise unlocks no additional concepts.

## Prerequisites

- `nothingness`
