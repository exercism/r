# Design

## Goal

The goal of this exercise is to introduce the student to function syntax, beyond what was discussed in `basics`.

## Learning objectives

- Understand the parts of a function, including optional and variable arguments (dot args).
- Understand the function environment and closures.
- Understand the potential of functions as return values.
- Know about anonymous function syntax.
- Be aware of Copy on Modify as a data-protecting default behavior.

## Out of scope

- Higher order functions, which are discussed in `functional-programming`.

## Concepts

The Concepts this exercise unlocks are:

- Functional Programming

## Prerequisites

- Vector Functions
- Nothingness
