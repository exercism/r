# Design

## Goal

The goal of this exercise is to teach the student how to use ICU-flavor regular expressions in R, with the `stringr` library.

## Learning objectives

- Understand that `stringr` uses the ICU library, which is similar to PCRE2 but not identical.
- Understand that String functions are often really Regex functions, such as `str_replace()`.
- Understand something about `str_replace()` syntax with captures.
- Understand `str_match`() and `str_match_all()` for captures, and how to unpack the (rather complcated) output format.

## Out of scope

- Teaching standard Regex syntax: _go online for that, we provide links_.

## Concepts

The Concepts this exercise unlocks are:

- `regular-expressions`.

## Prerequisites

- `strings`
- `functional-programming`
