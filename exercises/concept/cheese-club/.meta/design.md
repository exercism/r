# Design

## Goal

The goal of this exercise is to introduce the student to pipes, higher order functions, and modern functional-first design in R.

## Learning objectives

- Understand native pipes.
- Understand `lapply` and `sappply` in Base R.
- Get an overview of the `purrr` library and its benefits.

## Out of scope

- Higher order functions which return a function.

## Concepts

The Concepts this exercise unlocks are:

- regular-expressions
- dates-times
- dataframes

## Prerequisites

- lists
- loops
- functions
